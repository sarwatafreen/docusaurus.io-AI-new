# Lesson 4: Robot Walks Anywhere Without Crashing (Nav2)

In this lesson, we'll implement autonomous navigation using the Navigation2 stack, enabling our robot to safely navigate unknown environments without collisions. Navigation2 is the standard autonomous navigation framework for ROS 2.

## Prerequisites
- ROS 2 Humble Hawksbill
- Completed Module 3, Lessons 1-3
- Working VSLAM setup from Lesson 3
- Basic understanding of path planning algorithms

## What You'll Build
By the end of this lesson, you'll have:
1. Integrated Navigation2 with your robot platform
2. Configured obstacle avoidance and safety systems
3. Implemented autonomous path planning and execution
4. Created a complete navigation system for your humanoid

## Understanding Navigation2

Navigation2 is a collection of ROS 2 packages that provide:
- Global and local path planning
- Costmap management for obstacle avoidance
- Robot controllers for smooth motion
- Behavior trees for complex navigation behaviors
- Recovery behaviors for handling navigation failures

## Setting Up Navigation2

First, install Navigation2 packages:

```bash
# Install Navigation2 packages
sudo apt-get update
sudo apt-get install ros-humble-navigation2 ros-humble-nav2-bringup
```

# Launching the Navigation2 Stack

To launch the complete navigation stack with your robot:

```bash
# Source your ROS 2 workspace
source ~/your_workspace/install/setup.bash

# Launch the navigation stack
ros2 launch nav2_bringup navigation_launch.py \
  use_sim_time:=True \
  autostart:=True \
  params_file:=/path/to/navigation_params.yaml
```

# Running Navigation in Simulation

Once Navigation2 is running, you can send navigation goals:

1. Open RViz
2. Set the 2D Pose Estimate to localize your robot
3. Use the 2D Nav Goal tool to set destination
4. Watch your robot navigate autonomously!

# Testing Navigation Performance

To evaluate navigation performance, use the navigation performance evaluation tools:

```bash
# Record navigation performance
ros2 bag record /odom /cmd_vel /goal_pose

# Analyze navigation metrics
python3 analyze_nav_performance.py --bag_file nav_results.bag
```
